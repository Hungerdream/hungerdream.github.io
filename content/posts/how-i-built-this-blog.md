## 前言

最近花了点时间搭建了一个个人技术博客，选择了 Hugo 静态网站生成器 + Blowfish 主题的组合。

## 为什么选择 Hugo？

1. **速度快**：Go 语言编写，编译速度极快
2. **主题丰富**：社区提供了大量精美主题
3. **部署简单**：生成的静态文件可以直接部署到 GitHub Pages

## Blowfish 主题特点

- 🎨 现代化设计，支持亮色/暗色主题
- 📱 响应式布局，适配各种设备
- ⚡ 性能优秀，加载速度快
- 🔧 配置灵活，定制性强

## 部署到 GitHub Pages

1. 创建 GitHub 仓库
2. 推送 Hugo 项目
3. 配置 GitHub Actions 自动构建

## 总结

Hugo + Blowfish 是一个不错的博客搭建方案，推荐给想要快速搭建个人博客的开发者。
